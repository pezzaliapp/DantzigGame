Come si gioca a Dantzig Game

    
🎯 Scopo

    
Massimizza z = c₁·x + c₂·y muovendo il puntino giallo nella zona blu, rispettando i vincoli.

    
🕹️ Come si gioca

    
    
      
  1. Premi Nuovo per creare una mappa.
    2. 
      
  2. Trascina il puntino giallo dentro la zona blu.
    3. 
      
  3. Premi Verifica per confrontare il tuo valore con l’ottimo.
    4.
